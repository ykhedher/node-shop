# node-shop

This is a node shop RESTful API using:
## express
## JWT for authentification
## MongoDB & Mongoose


Clone

    Clone this repo to your local machine using https://github.com/ykhedher/node-shop.git

Setup


    update and install this package first

$ npm update
$ npm install 
$ node server.js
